Product pages

Posy's product page is built from reorderable blocks, so you decide what appears and in what order. Below the main section, two recommendation sections and a review-app host complete the default template.

Section settings

  • Image position — media gallery on the Left or Right of the product details.
  • Show low-stock countdown with Low-stock threshold — when the selected variant's inventory drops to the threshold or below, an "only X left" notice appears. It reads your real inventory and only that: with the setting off, or stock above the threshold, nothing is shown, and it never invents scarcity.
  • Add-ons heading — the heading above the "Complete the look" add-on products (see the Add-on product block below).
  • Show share buttons — social sharing links for the product.
  • Details section heading and Highlights — an at-a-glance list of product facts, one per line.

Blocks

Add, remove, and drag to reorder in the theme editor:

BlockWhat it does
Vendor & typeShows the product's vendor and product type from the product record.
TitleThe product title.
PricePrice and compare-at price, unit prices when set on a variant, and Shop Pay installments when enabled for your store.
Variant pickerOption pills, with swatches for color-like options. See "Swatches" below.
Buy buttonsQuantity selector and add-to-cart. Show additional checkout buttons toggles express payment buttons (Shop Pay, PayPal, Apple Pay…) based on what's enabled in Settings → Payments. When a product has subscription plans, the selling-plan selector appears here; on gift card products, the recipient form appears here.
DescriptionThe product description from the product record.
Highlight lineA one-line perk row (e.g. free shipping, warranty). Add several to build a perk list; each has a single Text setting.
Accordion itemA collapsible row with a Title and rich-text Text — ideal for shipping & returns, materials, or care instructions.
Add-on productAn optional add-on (e.g. a gift box) the shopper can tick to add alongside the main product. Pick the Product; multiple blocks build the "Complete the look" list.
App blocksBlocks provided by apps you've installed (size charts, wishlists, reviews…) can be added anywhere in the block order.

Media gallery

The gallery displays everything attached to the product in Products → [product] → Media: images, videos, external videos (YouTube/Vimeo), and 3D models, each with the appropriate player or viewer. When a variant has its own image, selecting that variant switches the gallery to it automatically.

Swatches

The Variant picker renders color-like options (Color, Finish…) as swatches instead of text pills. It supports Shopify's native swatch attributes — set a color or image per option value in Settings → Custom data or on the product's options — and falls back to a built-in library of common finish and color names when no native swatch is defined, so swatches work out of the box.

Pickup availability

If your store has local pickup configured (Settings → Shipping and delivery → Local pickup), the product page shows pickup availability for the selected variant, with a store list. No theme settings are needed — it appears automatically when pickup locations exist.

Product recommendations

The default template includes two instances of the same section, distinguished by Recommendation type:

  • "Pairs well with" — set to Complementary products. These come from the Search & Discovery app (free, by Shopify): install it and define complementary products per product. Until you do, this section stays empty — that is expected.
  • "You may also like" — set to Related products, generated automatically by Shopify from product data and behavior.

Each instance also has Eyebrow text, Heading, and Products to show settings.

Product reviews

The Loved by customers section is a host for review apps — Posy deliberately ships no built-in review widget, so your reviews always come from a real collection system:

  1. Install a reviews app (Judge.me, Loox, Air Reviews, or any app that provides theme blocks).
  2. In the theme editor, select the Loved by customers section on the product template and choose Add block — the app's review widget appears in the block list.
  3. Optionally set the section's Heading.
Seeing an empty section? Until an app block is added, the section renders nothing on your live store. That's by design — see the FAQ.

Featured product (section)

A home-page-friendly spotlight for a single product, with the same buy experience: pick the Product, choose Image position (Left / Right), and optionally enable Show vendor and Show additional checkout buttons. Use Eyebrow text, Heading, and the rich-text Text setting to frame it — Text replaces the product's own description when filled in, or leave it blank to use the product's description. App blocks are supported here too.
