Collections & search

Collection pages are where most shopping happens. Posy gives them flexible headers, faceted filtering with swatches, quick view, and an optional in-grid promo banner. The search page shares the same filtering.

Collection header

  • Header layout — Split (text beside the collection image), Centered, or Full bleed (text over the image).
  • Eyebrow text — the small line above the collection title.
  • Image badge text — a badge overlaid on the collection image.
  • Show rating with Overall rating and Review count — an optional rating line in the header. These fields are typed by you, so only display numbers backed by real reviews (off by default).

Product grid

  • Grid columns — 3 across or 4 across on desktop.
  • Products per page — 12 to 48; pagination appears beyond that.
  • Quick view — shoppers can open a product dialog from the grid to pick a variant and add to cart without leaving the collection.
  • Sorting — the standard Shopify sort orders (best selling, price, newest…).

Filters

Posy renders the filters you have enabled in the Search & Discovery app (Apps → Search & Discovery → Filters) — availability, price, and product options all work. Two theme settings shape them:

  • Enable variant filters — automatically builds a filter (Color, Size, Material, …) for every variant option used by products in the collection. Options whose name contains "color" or "finish" render as swatches, using the same swatch system as the product page.
  • Enable price filter — a dual-handle price range slider.

Active filters are shown as removable chips, with a clear-all link.

Promo banner block

An optional banner placed inside the product grid — useful for a seasonal offer or a collection story. Add up to two Promo banner blocks per collection section.

  • Position in product grid — which grid slot the banner occupies.
  • Eyebrow text, Heading, Text, Promo badge text — the banner copy. All blank by default: the banner shows only what you write.
Honest promos only A promo banner is advertising. Only announce offers that are real and current — an expired "20% off" banner is the fastest way to lose a customer's trust (and can violate consumer protection rules).

Collections list page

The Collections grid section powers your /collections page, listing every collection with Eyebrow text, Heading, Subheading, and Collections per page settings.

Search page

The search results page groups matches by products, pages, and articles, and offers the same filtering and sorting as collection pages for product results. Most shoppers will get there through the header's predictive search overlay; the page itself is the full-results and no-JavaScript fallback.
