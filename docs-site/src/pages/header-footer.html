Header & footer

The header and footer are section groups — they appear on every page of your store and are edited once, from any page in the theme editor.

Announcement bar

A slim bar above the header for shipping offers, launches, or store news. Add up to five Message blocks; when there is more than one, messages rotate automatically with a gentle fade.

Each message has a single Text setting.

Edit the sample messages The bar ships with sample copy such as "Complimentary gift wrap on every order." Replace the samples with offers that are true for your store — or remove the blocks you don't need.

Header

The header holds your logo, navigation, search, account, and cart controls. Its settings:

  • Menu — the main navigation menu. Defaults to your store's Main menu; build and reorder menus in Shopify admin → Content → Menus. Menu items with sub-items open as dropdowns.
  • Customer account menu — an optional menu shown to logged-in customers. Leave it empty to use the standard account links. Posy uses Shopify's hosted customer accounts, so login, orders, and profile pages are handled by Shopify and always up to date.

Cart drawer

Selecting the bag icon opens a sliding cart drawer instead of navigating away, so shoppers can keep browsing. The drawer shows line items with quantity controls, the subtotal, and a checkout button — and if you have set a Free shipping threshold in Theme settings → Cart, a live progress bar toward free shipping. See Cart.

Predictive search

Selecting the search icon opens a search overlay. As the shopper types, results appear live, grouped into suggestions, products, and pages & collections. Pressing Enter goes to the full search results page. If JavaScript is unavailable, the search field falls back to the standard search page — no dead ends.

Footer

The footer carries your brand sign-off, navigation columns, and store-level utilities.

Settings

  • Brand tagline — a short line under your store name.
  • Instagram URL, TikTok URL, Pinterest URL — social icons appear only for the URLs you fill in; leave one empty and its icon is hidden.
  • Show payment icons — displays the payment methods enabled for your store.
  • Show country/region selector and Show language selector — these appear only when your store has more than one country/currency or language enabled. See Localization.
  • Show Follow on Shop button — lets shoppers follow your store in the Shop app. Requires the Shop sales channel.
  • Copyright text — the line printed next to the year and your store name.

Link columns

Add up to four Link column blocks. Each block has a Heading and a Menu picker — create small menus (Shop, Help, About…) in Content → Menus and assign one per column.
