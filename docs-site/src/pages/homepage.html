Home page sections

Posy's home page ships with six sections in this order: Hero, Featured collections, Best sellers, Reviews, Instagram gallery, and Newsletter. Add, remove, and reorder them freely in the theme editor — this page documents each one.

Hero

The opening statement of your store. Three layouts are available under Hero layout:

  • Split — text beside a single large image.
  • Centered — centered text between two images (Image and Second image (centered layout)).
  • Editorial — a full-width, magazine-style composition with an Editorial tag (top corner).

Content settings: Eyebrow text, Heading (supports italics for an accent word), Subheading, plus two calls to action — Primary button text / Primary button link and Secondary button text / Secondary button link.

Stats and chips

With Show stats enabled, up to three stat pairs (Stat 1 — value / Stat 1 — label, and so on) render under the hero text, and two floating chips (Floating chip text, Floating review chip text) can decorate the image. All of these are blank by default — they appear only when you write them, and they should state facts that are true for your store.

Featured collections

A showcase of hand-picked collections. Settings: Eyebrow text, Heading, and Collections — pick the collections to feature and their order.

Best sellers

A product grid pulled from a collection of your choice.

  • Collection — the source collection (a "best sellers" collection works well, but any collection can power it).
  • Collection filters — optionally pick additional collections to show as filter chips above the grid, letting shoppers flip between them without leaving the page.
  • Number of products — show 4, 8, or 12 products.
  • View all button text and View all button link — the button below the grid.

Reviews

A section for quoting customer reviews, with Eyebrow text and Heading settings and up to six Review blocks. Each block has Reviewer name, Reviewer handle, Star rating, Text, and a Verified buyer checkbox.

Real reviews only These quotes are written by you, so treat them like advertising: quote real customers, and tick Verified buyer only for reviews that come from actual purchases. If you want automatically collected reviews on product pages, see the product reviews section, which hosts review apps.

Instagram gallery

A curated grid of up to six Post blocks, headed by your Hashtag, Heading, and Instagram handle. Each post has an Image, a Like count, and a Link URL (link each tile to the matching post or to your profile).

This is a curated grid, not a live feed You choose the images and type the like counts yourself — nothing is pulled from Instagram automatically, so the section works without connecting an account or an app. Keep the numbers honest and refresh the grid now and then so it matches your actual profile.

Newsletter

An email sign-up form wired to Shopify: subscribers land in Customers with their marketing status set, ready for Shopify Email or any connected email app.

  • Heading, Subheading, Button text — the pitch. The defaults advertise a 15% welcome offer; edit them to match what you actually offer.
  • Success message — shown after sign-up. The default mentions a sample discount code (HELLO15), so replace it with your own code — or remove the mention — before launch.
  • Fine print — the small reassurance line under the form.

More sections to add

Beyond the defaults, the Add section picker also offers: Featured product, FAQ accordion and FAQ contact CTA, Store locations, the About and Contact sections, and two power-user tools — Custom section and Custom Liquid.
