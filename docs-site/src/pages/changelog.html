Changelog

Release notes for every version of Posy. After updating, review this page to see what changed.

1.0.0 — Initial release

Posy's first public release on the Shopify Theme Store.

  • Four theme styles: Posy, Amethyst, Meadow, and Sunbeam.
  • Home page sections: Hero (three layouts), Featured collections, Best sellers, Reviews, Instagram gallery, Newsletter.
  • Block-based product page with media gallery (video, external video, 3D models), swatches, selling-plan support, add-on products, low-stock countdown, and pickup availability.
  • Collections with faceted filters, swatch filtering, quick view, and in-grid promo banners.
  • Cart page and drawer with free-shipping progress bar, promo codes, and express checkout.
  • About, Contact (with store locations map), and FAQ page templates.
  • Full localization support: markets, currencies, languages, and styled gift cards with recipient form.
How Posy updates work When a new version reaches the Theme Store, Shopify notifies you in Online Store → Themes. Updates install as a new copy of the theme, so your published store is never changed without you — preview the updated copy, re-apply any code customizations, then publish.
